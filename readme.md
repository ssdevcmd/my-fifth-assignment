    - 3️⃣ What is the difference between map(), filter(), and forEach()?

    Ans: map() is used for transform data, it will works on every single data and gives an new array as an output.

    filter() is used for select items. If the condition match then it will return every items.

    forEach() is used to loop through in every items but it will return nothing.
